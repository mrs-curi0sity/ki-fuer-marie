### Voraussetzungen
- Python 3.9+
- Poetry
- VS Code + Marp Extension (für Slides)

### Installation
1. Repository klonen:
```bash
git clone git@github.com:mrs-curi0sity/ki-fuer-marie.git
cd ki-fuer-marie
```

2. Poetry-Umgebung einrichten:
```bash
poetry install
```
